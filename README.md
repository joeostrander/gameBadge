
# gameBadge

**Development changes**:  
Examples in this repo are set to use Visual Studio Code with the PlatformIO extension.  Uploading is done via UPDI using pymcuprog.  
```bash
pip install pymcuprog
```
**PCB Rev C changes**:  
-UPDI pinout  
-lm1117 regulator  
-micro USB  
-smaller battery  
-12mm buzzer  
-SMD LEDs
